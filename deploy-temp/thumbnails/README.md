# Project Thumbnails Folder

This folder contains thumbnail images for the featured projects section.

## Naming Convention
- `sentinel-ai.jpg` - Sentinel AI - Disaster Early-Help Platform
- `mental-health-chatbot.jpg` - AI-Powered Mental Health Chatbot
- Add more project thumbnails as needed

## Image Specifications
- **Recommended Size:** 500x300px or 600x400px
- **Format:** JPG, PNG, or WebP
- **Max Size:** 200KB per image for performance

## How to Add
1. Save your thumbnail image with a descriptive name
2. Place it in this folder (`public/thumbnails/`)
3. Reference in your code as: `/thumbnails/filename.jpg`

Example in React/HTML:
```html
<img src="/thumbnails/sentinel-ai.jpg" alt="Sentinel AI" />
```
